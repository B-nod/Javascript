Regular Expressions

/../-> regular expression must write in the dotted line
^ -> it checks from the begining
$ -> it checks from the end
[a-z] -> matches only lowercase alphabets
[A-Z] -> matches only uppercase alphabets
[a-zA-Z] -> matches both the uppercase and lowercase alphabets
[0-9] -> matches only numeric value

for special character use \

example: \@,\-,\_,\. etc

* -> matches one or more preceeding character
. -> matches single character
?=. -> this symbol denotes that position of any string or character doesnot 
       need to be at any specific position whether it can come in the begining or last.
